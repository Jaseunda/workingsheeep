# Contributing

Contributions are welcome.

This repository is not open source. It is a public asset repository for the Sheeep project, and all rights remain with the owner.

## What You Can Contribute

- new screenshots
- cleaner exports of existing screenshots
- updated marketplace assets
- README improvements
- asset organization fixes

## Before Opening a Pull Request

- keep filenames stable unless there is a good reason to change them
- prefer optimized image sizes without visibly hurting quality
- avoid changing public asset URLs unnecessarily
- describe what changed and why

## Rights

By submitting a contribution, you agree that the owner may use, modify, relicense, and distribute your contribution as part of this repository and the Sheeep project without further approval or compensation.

If you do not agree to those terms, do not submit a contribution.
